Lesson 3 Exercises - Strings

  1. Create the text 'My name is:' as a string.
  2. Create a name variable (for example: Simon).
  3. Using a template string add the name variable to the 'my name is'.
  4. At a restaurant, you order 1 coffee ($5) and a bagel ($3). Using a template string calculate the total cost.
  5. Create a multi-line template string.
